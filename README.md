# producerAgent

Markdown based agent.

This repository contains a simple scaffold for building an agent that uses Markdown files as input/output.

## Structure

- README.md — this file
- .gitignore — files to ignore
- agent/ — place for agent source files and markdown-based prompts

## Getting started

1. Add your agent implementation in the `agent/` directory.
2. Write markdown prompts or configuration files that the agent will consume.
